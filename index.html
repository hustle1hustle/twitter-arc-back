🐦 Twitter Arc API

Backend API for Twitter reputation scoring and analytics.

Available Endpoints:

📊 Reputation Score

GET /.netlify/functions/rep?u=username

Get reputation score and analytics for a Twitter user.

🔐 OAuth Authentication

GET /.netlify/functions/auth_x

Twitter OAuth flow for user authentication.

🧪 Test Function

GET /.netlify/functions/test-simple

Simple test endpoint to verify functions are working.

Example Usage:

curl "https://your-site.netlify.app/.netlify/functions/rep?u=jack"

Built with Netlify Functions & TypeScript
